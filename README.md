# Data Science Projects

This repository contains a collection of data science and machine learning projects showcasing data analysis, modeling, and visualization skills.

---

## Projects

### 1. House Price Prediction
- Built a machine learning model to predict housing prices in Bengaluru
- Performed data cleaning, feature engineering, and outlier removal
- Achieved ~18% error using MAE as evaluation metric  

Folder: `House-Price-Prediction`

---

### 2. Instrinsic Valuation DCF & Monte Carlo Simulation
- Developed a discounted cash flow (DCF) model for stock valuation
- Applied Monte Carlo simulation to account for uncertainty in growth and discount rates
- Performed sensitivity analysis to evaluate valuation ranges  

Folder: `Instrinsic-valuation-DCF-Monte-Carlo`

---

## Tech Stack
- Python (NumPy, Pandas, Scikit-learn)
- Data Visualization (Matplotlib, Plotly)
- Jupyter Notebook
