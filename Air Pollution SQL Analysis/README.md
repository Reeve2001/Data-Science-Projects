# Air Pollution Trend Analysis Across Indian Cities (2015–2024)

## Overview
This project analyzes 10 years of air pollution and AQI data across major Indian cities using SQL. The analysis focuses on pollution trends, seasonal patterns, AQI category distribution, and identifying high-risk pollution periods.

---

## Skills Demonstrated
- SQL
- Common Table Expressions (CTEs)
- Window Functions
- Ranking Functions
- Aggregate Functions
- CASE Statements
- Time-Series Analysis

---

## Key Analyses
1. Average AQI per city
2. Yearly AQI trends
3. Most polluted month for each city
4. AQI category distribution
5. Most improved city over time
6. Top 5 worst pollution days
7. Seasonal pollution analysis

---

## Tools Used
- MySQL
- GitHub
